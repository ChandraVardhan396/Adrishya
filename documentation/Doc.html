Adrishya AI - Image Object Remover

Adrishya AI is a Java-based IntelliJ project that removes unwanted objects from images without quality loss. It uses Object-Oriented Programming concepts, a sleek FlatLaf UI, MySQL database, and the LAMA AI model via ONNX Runtime for fast, accurate, and seamless object removal.

Features
  • Lossless removal of unwanted objects from images
  • Java backend built with Object-Oriented Programming (OOP) principles
  • Modern FlatLaf UI for a smooth user experience
  • MySQL database for efficient data management
  • Powered by LAMA AI model using ONNX Runtime for precise editing
  • Developed as an IntelliJ IDEA project

Setup & Run Instructions
  1. Ensure Python with pip is installed on your system.
  2. Download the LAMA AI model from the link below:
    Download LAMA AI Model
  3. Place the downloaded model file in the src directory, in the same folder as MainApp.java.
  4. Open your terminal or command prompt and navigate to the project directory:
    cd path/to/AdrishyaAI
  5. Install the required Python packages:
    pip install -r requirements.txt
  6. Open the project in IntelliJ IDEA.
  7. In IntelliJ, open MainApp.java and run the project. The GUI window will open.
  8. Load the image you want to edit.
  9. Use the Lasso Selection tool to easily select the object you want to remove.

Note: Install the latest Python version and ensure pip is enabled during installation.

Project Structure & SDK Setup

Follow these steps to configure your IntelliJ project SDK properly:

  1. Open Project Structure:
    File → Project Structure (or press Ctrl + Alt + Shift + S)
  2. Go to Project Settings > Project:
    • Under Project SDK, make sure a valid JDK (e.g., JDK 21 or JDK 213) is selected.
    • If nothing is selected or it shows “<No SDK>”, you must add one.
    • To add a JDK:
      Click New... → JDK, then browse to the folder where your JDK is installed (e.g., C:\Program Files\Java\jdk-23).
    • Select it and click Apply.
  3. Go to Project Settings > Modules:
    • Select your module (e.g., ai_object).
    • Under Dependencies, ensure the Module SDK is set to the JDK you just added (not JDK 0).
    • If it's set to inherited from project, that’s fine as long as the project JDK is valid.
  4. Click Apply then OK to save your settings.
